## 📋 What is catchclaw?

catchclaw is a tool that helps check the safety of AI platforms. It works on Windows and other systems. It supports many AI platforms like OpenClaw, Dify, FastGPT, and more. The tool scans for weaknesses using 78 attack paths and modules. It also maps these to different stages of attack techniques. 

catchclaw uses a fast engine to run checks and shows the results clearly with visuals. It finds vulnerabilities before attackers do. You do not need to be a security expert to use it.
